# TeamAI - Экраны Авторизации

## Реализованные экраны

### 1. Login Screen (`lib/screens/auth/login_screen.dart`)

**Функциональность:**
- Вход по email и паролю
- Валидация полей формы
- Показ/скрытие пароля
- Ссылка "Forgot password?"
- Вход через Google
- Переход к регистрации через "Sign Up"

**Элементы UI:**
- Логотип TeamAI и подзаголовок "Management App"
- Заголовок "Войдите в свой аккаунт"
- Поле Email с иконкой
- Поле Password с иконкой и кнопкой показа/скрытия
- Синяя кнопка "Login"
- Разделитель "Or Login with"
- Кнопка входа через Google
- Ссылка "Don't have an account? Sign Up"

### 2. Register Screen (`lib/screens/auth/register_screen.dart`)

**Функциональность:**
- Регистрация с username, email и паролем
- Подтверждение пароля
- Валидация всех полей
- Показ/скрытие паролей
- Регистрация через Google
- Кнопка "Назад" для возврата

**Элементы UI:**
- Кнопка "Назад" в верхнем левом углу
- Логотип TeamAI и подзаголовок "Management App"
- Заголовок "Создайте свой аккаунт"
- Поле Username с иконкой
- Поле Email с иконкой
- Поле Password с иконкой и кнопкой показа/скрытия
- Поле Confirm Password с иконкой и кнопкой показа/скрытия
- Синяя кнопка "Register"
- Разделитель "Or Register with"
- Кнопка регистрации через Google

## Навигация

```
Splash Screen (3 сек)
    ↓
Onboarding Screen (3 страницы)
    ↓ (Продолжить / пропустить)
Login Screen
    ↔ (переход между экранами)
Register Screen
```

## Валидация форм

### Login:
- Email: обязательное, должно содержать @
- Password: обязательное

### Register:
- Username: обязательное
- Email: обязательное, должно содержать @
- Password: обязательное, минимум 6 символов
- Confirm Password: должен совпадать с паролем

## Цветовая схема

- **Primary Blue**: `#4F9CF9` - кнопки, активные элементы, ссылки
- **Background White**: `#FFFFFF`
- **Input Background**: `#F5F5F5`
- **Text Primary**: `#212121`
- **Text Secondary**: `#757575`
- **Text Placeholder**: `#BDBDBD`
- **Icon Background**: `#E0E0E0`
- **Border**: `#E0E0E0`

## Особенности UI

### Поля ввода:
- Синяя иконка слева (50x56px, radius 12)
- Светло-серый фон поля (radius 12)
- Hint текст серого цвета
- Кнопки показа/скрытия пароля

### Кнопки:
- Основные кнопки: синий фон, белый текст, radius 12, высота 56px
- Текстовые кнопки: синий текст без фона

### Социальная авторизация:
- Квадратная кнопка (64x64px) с иконкой Google
- Белый фон с серой рамкой
