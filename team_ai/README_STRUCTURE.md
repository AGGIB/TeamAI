# TeamAI - Структура проекта

## Описание
Мобильное приложение для интеллектуального управления командной работой с использованием AI.

## Реализованные экраны

### 1. Splash Screen (`lib/screens/splash_screen.dart`)
- Отображает логотип TeamAI по центру экрана
- Автоматически переходит к онбордингу через 3 секунды
- Использует SVG иконку: `assets/logo_teamAI.svg`

### 2. Onboarding Screen (`lib/screens/onboarding_screen.dart`)
Три страницы онбординга с возможностью:
- Пролистывания между страницами
- Пропуска онбординга (кнопка "skip")
- Индикаторы текущей страницы
- Кнопка "Get Started" для перехода к следующей странице

#### Страницы:
1. **Автоматическое Распределение Задач**
   - Иллюстрация: `assets/first.svg`
   - Описание AI-распределения задач

2. **Максимизация Эффективности Команды**
   - Иллюстрация: `assets/second.svg`
   - Описание анализа навыков участников

3. **Своевременные Уведомления и Контроль**
   - Иллюстрация: `assets/third.svg`
   - Описание системы уведомлений

## Цветовая схема
- Основной цвет (Primary): `#4F9CF9` (голубой)
- Фон: `#FFFFFF` (белый)
- Текст заголовков: `#212121` (темно-серый)
- Текст описаний: `#757575` (серый)
- Неактивные элементы: `#E0E0E0` (светло-серый)

## Зависимости
- `flutter_svg: ^2.0.10+1` - для работы с SVG иллюстрациями

## Assets
Все SVG иллюстрации находятся в папке `assets/`:
- `logo_teamAI.svg` - логотип приложения
- `first.svg` - иллюстрация первой страницы онбординга
- `second.svg` - иллюстрация второй страницы онбординга
- `third.svg` - иллюстрация третьей страницы онбординга

## Запуск проекта
```bash
# Установить зависимости
flutter pub get

# Запустить приложение
flutter run
```
