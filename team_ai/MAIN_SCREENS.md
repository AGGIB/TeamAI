# TeamAI - Основные экраны приложения

## Структура приложения

```
lib/screens/
├── main_screen.dart              # Главный контейнер с Bottom Navigation
├── home/
│   └── home_screen.dart         # Домашняя страница
├── ai_agent/
│   └── ai_agent_screen.dart     # AI Агент для распределения задач
├── calendar/
│   └── calendar_screen.dart     # Календарь и Kanban доска
└── profile/
    └── profile_screen.dart      # Профиль пользователя
```

## 1. Main Screen (Главный контейнер)

**Файл:** `lib/screens/main_screen.dart`

**Функционал:**
- Bottom Navigation Bar с 4 вкладками
- Переключение между экранами
- SVG иконки для навигации
- Активная вкладка подсвечивается синим цветом

**Bottom Navigation элементы:**
1. **Home** - `assets/bottomNav/Home.svg`
2. **AI Agent** - `assets/bottomNav/ai_agent.svg`
3. **Calendar** - `assets/bottomNav/calendar.svg`
4. **Profile** - `assets/bottomNav/user_account.svg`

---

## 2. Home Screen (Главная страница)

**Файл:** `lib/screens/home/home_screen.dart`

**Элементы UI:**

### Header
- Дата: "Среда, 15 ноября" (динамически)
- Иконка уведомлений (синий фон)

### Приветствие
- Динамическое приветствие по времени суток:
  - Утро (до 12:00): "Доброе утро"
  - День (12:00-18:00): "Добрый день"
  - Вечер (после 18:00): "Добрый вечер"
- Имя пользователя: "[Имя Пользователя]!"
- Индикатор загруженности: "Ваша загруженность: 65%"

### Мои задачи
- Горизонтальная прокрутка карточек проектов
- Каждая карточка содержит:
  - Бейдж с количеством дней
  - Название проекта/задачи
  - Подзаголовок с описанием
  - Прогресс-бар
  - Цветной фон (синий, фиолетовый, красный)

### Мои Задачи на Сегодня
- Список задач с чекбоксами
- Выполненные задачи отмечены синим
- Невыполненные - обычный круг с рамкой

---

## 3. AI Agent Screen

**Файл:** `lib/screens/ai_agent/ai_agent_screen.dart`

**Функционал:**
- Экран для AI-функций распределения задач
- Управление командой
- Аналитика производительности

**Элементы:**
- Заголовок "AI Агент"
- Подзаголовок "Умное распределение задач"
- 4 карточки функций:
  1. Автоматическое распределение (синий)
  2. Управление командой (фиолетовый)
  3. Аналитика производительности (бирюзовый)
  4. Умные рекомендации (оранжевый)
- Бейдж "Скоро..." внизу

---

## 4. Calendar Screen

**Файл:** `lib/screens/calendar/calendar_screen.dart`

**Функционал:**
- Календарь с дедлайнами
- Kanban доска для управления задачами

**Элементы:**

### Календарь (placeholder)
- Область для отображения календаря
- Иконка календаря

### Kanban Доска
3 колонки:
1. **К выполнению** (синий) - 5 задач
2. **В работе** (оранжевый) - 3 задачи
3. **Завершено** (зеленый) - 8 задач

Каждая колонка:
- Название
- Счетчик задач
- Область для карточек задач

---

## 5. Profile Screen

**Файл:** `lib/screens/profile/profile_screen.dart`

**Элементы:**

### Профиль пользователя
- Аватар (круглый, синий фон)
- Имя: "[Имя Пользователя]"
- Должность: "Full Stack Developer"

### Статистика
3 карточки:
- **15** Задач
- **3** Команды
- **92%** Эффективность

### Навыки (Skills)
Chip-элементы:
- Flutter, Dart, React, Node.js, Python, UI/UX Design

### Мои команды
Список команд:
1. **Team Alpha** - 12 участников (синий)
2. **Design Squad** - 8 участников (фиолетовый)
3. **Backend Team** - 15 участников (бирюзовый)

### Настройки
Список опций:
- Уведомления
- Конфиденциальность
- Язык
- Помощь
- Выйти (красный цвет)

---
